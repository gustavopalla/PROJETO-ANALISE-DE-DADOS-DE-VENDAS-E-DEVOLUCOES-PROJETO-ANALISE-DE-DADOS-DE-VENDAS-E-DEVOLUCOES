PROJETO: ANÁLISE DE DADOS DE VENDAS E DEVOLUÇÕES

Este projeto tem como objetivo ler, interpretar e analisar dados de vendas a partir de arquivos .csv. 
Os dados contêm informações sobre produtos vendidos e devoluções realizadas. Após o processamento, o 
programa organiza as informações e gera gráficos interativos que facilitam a visualização de:

- Produtos mais vendidos;
- Faturamento por produto e por loja;
- Quantidade de produtos devolvidos.

A análise oferece uma visão clara e rápida sobre o desempenho comercial com base nos dados dos arquivos
